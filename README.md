# Digital Portfolio — Editorial Aesthetic

Modern, minimalist, and editorial-inspired portfolio built with a focus on typography, motion, and accessibility.

## 🚀 Tech Stack

- **Core**: [React](https://react.dev/) + [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/) + [SCSS](https://sass-lang.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Smooth Scroll**: [Lenis](https://lenis.darkroom.engineering/)
- **Navigation**: [React Scroll](https://www.npmjs.com/package/react-scroll)

## ✨ Key Features

- **Editorial Design**: Asymmetrical grids and high-contrast typography (Serif for headings, Sans-serif for body).
- **Dynamic Dark Mode**: Seamless theme switching using CSS Variable Orchestration (persisted in `localStorage`).
- **Smooth Scrolling**: High-quality inertia scrolling integrated with Lenis.
- **Micro-animations**: Staggered text reveals and subtle scroll-triggered transitions.
- **Accessibility**: Semantic HTML, ARIA labels, and `focus-visible` states.
- **Performance**: Optimized builds and lazy-loaded assets.

## 🛠️ Installation & Setup

1. **Clone the repository**:
   ```bash
   git clone <your-repo-url>
   cd and318-spa
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run development server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```

## 📂 Project Structure

- `src/components/`: Modular UI components (Navbar, Hero, PortfolioGrid, etc.).
- `src/styles/`: Global styles and Tailwind configuration.
- `src/pages/`: Main page layouts.
- `public/`: Static assets.
